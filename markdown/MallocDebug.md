

Utility for monitoring memory allocation using malloc() and free() calls. Can be used to help track down all kinds of General/MemoryLeaks. 

See also General/ObjectAlloc for tracking the history of General/ObjC objects.

Apple have a short explanation on how to use General/MallocDebug at http://developer.apple.com/documentation/Performance/Conceptual/General/ManagingMemory/Articles/General/FindingLeaks.html
