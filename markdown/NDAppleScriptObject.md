


* **Name:** General/NDAppleScriptObject
* **Description:** Wrapper to General/AppleScript.
* **URL:** http://homepage.mac.com/nathan_day/
* **Author:** Nathan Day.
* **Functionality present:** 
* Compiling and executing General/AppleScript fragments
* Getting General/NSData representing compiled scripts
* Writing scripts to file
* More... 
* **Functionality missing:** 
* Pretty much nothing. 
* **Notes:** 
* In 10.2+, you might as well just use General/NSAppleScript. However, General/NDAppleScriptObject provides a lot of additional functionality not provided by General/NSAppleScript.
