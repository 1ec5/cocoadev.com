

The General/AppKit is a set of classes, first developed by General/NeXT Computer, and now by General/AppleComputer, for rapidly developing applications. The majority of General/AppKit classes deal with user interface elements, such as windows, buttons, and menus. General/InterfaceBuilder lets you work with many of these classes visually.

The General/AppKit builds upon General/FoundationKit, and thus requires it to function.

Note that the General/AppKit classes are generally _not_ thread safe.
