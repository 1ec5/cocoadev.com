Here are some conditional statements that are used often in C and Objective-C


----
The if conditional statement
----
<code>
if (i<10)

{

}
</code>
----
The if-else conditional statement
----
<code>
if (i<10)

{

}

else

{

}
</code>
----
The switch conditional statement
----
<code>
switch()

{

     case (1): blah, blah;

                   break;

     case (2): blah, blah;

                   break;

     default:  blah, blah;

                   break;

}
</code>
